# MuscleQuest

## Installed packages

- React Native Paper
- React Hook Form
- Yup
- React Native Async Storage
- React Native Firebase
- Expo Secure Store
- Tanstack Query
- Expo Notifications
- Expo Image Picker
- date-fns (Date and Time)
- Expo SQLite
- Sentry

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
